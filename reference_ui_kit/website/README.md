# VILT Marketing Website — UI Kit

A high-fidelity recreation of the VILT Group corporate marketing website (`vilt-group.com`) as a clickable React/JSX prototype.

**Components**
- `TopNav` — sticky nav with primary links + language switcher + contact CTA
- `LangSwitcher` — 6-language dropdown (EN/ES/PT-PT/PT-BR/DE/FR)
- `HeroBlock` — split layout: bold mixed-weight headline + warm photo-led illustration
- `IntroBanner` — orange "Apresentando a VILT >" pill that floats top-right
- `SectionEyebrow` — `eyebrow` typographic style; precedes h2
- `TXBlock` — Total Experience explainer with CX / EX tabs
- `DiffCard` — "what makes us unique" rounded photo + title + body
- `CommandmentList` — numbered "5 Mandamentos" list
- `JoinUsCTA` — orange "joinus+" recruitment band with rotating "+"
- `LetsChatForm` — contact form (name / email / subject / message + checkbox)
- `Footer` — deep-violet footer with logo, blurb, certifications, legal links

**Behaviour**
- Tabs in the TX block switch between CX / EX content
- Hover states on nav links + cards (lift + accent colour)
- Form inputs focus with orange ring
- Language switcher opens a popover

**Coverage scope**
This kit recreates the *visual style + interaction patterns* of the homepage above the fold and the about-us hero. It is not a full site recreation (industries, insights, partners pages omitted).
